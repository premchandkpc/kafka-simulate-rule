## Principles

- Specification-first. Language-agnostic. Multiple implementations.
- The spec is the primary artifact. Code is secondary.
- Every subsystem defined by stable contracts, interfaces, and protocols.
- Compile once. Execute forever. Immutable bytecode. Deterministic execution.
